# OpenPUA

## Description

OpenPUA est une skill open source conçue pour améliorer la persévérance et la rigueur des agents IA lors des tâches complexes de débogage.

Au lieu d'accepter rapidement un échec ou une impasse, OpenPUA impose une méthodologie structurée qui pousse l'agent à :

- Vérifier systématiquement les journaux d'erreurs
- Explorer plusieurs pistes de résolution
- Consulter la documentation pertinente
- Tester différentes hypothèses
- Identifier la cause racine du problème
- Éviter les boucles de raisonnement répétitives.

Le projet applique une approche d'escalade progressive et des checklists obligatoires afin de réduire les abandons prématurés lors des investigations techniques.

## Fonctionnalités principales

### Escalade progressive

Système de plusieurs niveaux visant à empêcher les abandons précoces et à encourager l'exploration de solutions alternatives.

### Checklist obligatoire

Validation systématique de plusieurs éléments avant de conclure à un échec :

- Logs
- Dépendances
- Configuration
- Documentation
- Hypothèses alternatives
- Validation finale

### Méthodologie de débogage

Approche structurée :

1. Reproduire
2. Isoler
3. Formuler une hypothèse
4. Tester
5. Vérifier
